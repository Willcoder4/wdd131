Goal Tracker - Site Plan

Site Name:

GrowNow

Site Purpose:

The site provides educational content and a demo goal tracking app to help people know how to set and track their goals for success and growth.

Scenarios:

  • What is goal setting?
  • What is goal tracking?
  • How can I set and track my goals for success?

Color Schema:

  • Primary Color - #2D6A4F: Used for headings, icons and primary action buttons
  • Secondary Color - #F8F9FA: Used for background
  • Accent Color - #212529: Used for body texts

Typography:

  • Montserrat: used on the app icon, dashboard title, goal titles and educational headers (h1, h2, h3)
  • Inter (Regular): Used on the paragraph texts.
  • Inter (Medium/Semi-Bold): Used on tracking data and function UIs.
